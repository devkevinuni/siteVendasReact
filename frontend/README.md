# React + Vite

This project is a frontend-only React application built with Vite and JavaScript.

Features:
- React 19 with JSX support
- React Router DOM for client-side routing
- Tailwind CSS for styling
- ESLint configured for `.js` and `.jsx` files

## Development

Use the following commands from the `frontend` folder:

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
